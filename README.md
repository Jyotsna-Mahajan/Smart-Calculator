# 🧮 Smart Calculator (CLI-Based)

A Python-based advanced command-line calculator that evaluates mathematical expressions and stores calculation history both in memory and in a file.

## 🚀 Features

- ➕ Evaluate mathematical expressions (e.g., `5 + 3 * 2`)
- 📂 Stores history in a file (`history.txt`)
- 📜 View previous calculations
- 🗑️ Clear history (both memory and file)
- 🔁 Menu-driven CLI interface
- ⚠️ Handles invalid input gracefully


## 🧠 Concepts Used

- Python Functions  
- Loops (`while`)  
- Conditional Statements  
- Exception Handling (`try-except`)  
- File Handling (read/write/append)  
- Lists (for storing history)  
- Built-in function: `eval()`
